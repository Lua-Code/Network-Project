Quick Steps to Run the UDP Test

1.Open WSL Ubuntu

-If you're using windows download WSL and use Ubuntu

2.Navigate to the test folder

cd /mnt/<drive_letter>/<path_to_folder>


3.Make the script executable 

chmod +x run_udp_test.sh


4.Run the test

./run_udp_test.sh


5.Enter your sudo password when prompted.
